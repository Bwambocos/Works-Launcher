＊起動方法
	・32ビットOSの方は「AC_Dash (32bit).exe」、64ビットOSの方は「AC_Dash (64bit).exe」をダブルクリックし起動してください。

＊操作方法
	・キーボードとマウスを使って操作できます。詳しくは、初回プレイ時のチュートリアルをご覧ください。
	・終了はメニュー画面から「終了する」をクリックしてください。この方法以外で終了した場合の不具合について作者は一切責任を負いません。

＊動作環境
	・Windows環境で正常に起動します。
	・Windows 10, Windows Server 2012 R2 Datacenter で動作を確認しています。
	・なお、MacOS 環境においては特別なツールを使用しない限り起動できませんのでご注意ください。
